# klock
Simple kivy clock app

## usage

```bash
$ git clone https://github.com/Anzo52/klock && cd klock
$ python -m venv env
$ source env/bin/activate
$ pip install -r requirements.txt
$ python klock.py
```
If installing the requirements file doesn't work, try:
```bash
$ pip install kivy[full]
```
